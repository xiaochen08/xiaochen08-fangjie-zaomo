## 图片最低要求

- 横版图片建议 16:9；
- 模型轮廓、关节和纹理能看清；
- 不用烟雾和强光遮住结构；
- 不出现 Blockbench 做不到的连续曲面；
- 中文标题清楚，没有乱码；
- 同一套图片保持黑铁、青蓝能量和暗黑游戏风；
- 图片压缩后仍然清晰。

## 上传后记得检查

1. GitHub 能正常显示图片；
2. 文件名大小写完全一致；
3. README 和网页没有引用错图；
4. 手机查看时文字仍然能看清。

图片由你亲自选择和上传，保证公开展示的是你认可的版本。Enjoy! ✨
